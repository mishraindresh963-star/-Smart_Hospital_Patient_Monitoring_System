# Smart Hospital Patient Monitoring - Fullstack Scaffold

This archive contains a minimal working scaffold for:
- Backend: Node.js + Express + MongoDB (Mongoose)
- Frontend: React (simple components)

See `/backend` and `/frontend` folders for details.

## Quick start (local)
1. Start MongoDB locally (or use MongoDB Atlas).
2. Backend:
   - cd backend
   - copy .env.example to .env and edit
   - npm install
   - npm run dev
3. Frontend:
   - cd frontend
   - npm install
   - npm start
